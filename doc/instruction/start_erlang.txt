ISTRUZIONI PER AVVIARE ERLANG:

1)Aprire tanti terminali quanti i nodi ed eseguire rispettivamente
	erl -sname NomeNodoDispather1
	...
	erl -sname NomeNodoDispatcherN
	erl -sname NomeNodoExecutor1
	...
	erl -sname NomeNodoExecutorM

	Pattern NomeNodoDispatherX:
		dispX@localhost
	Pattern NomeNodoExecutorY:
		execY@localhost
	dove X va da 1 a N, y va da 1 a M

	
	A questo punto non serve più creare il nodo erlang auction_manager perché java
	lo creerà da solo. Il problema di creazione del nodo per Java avviene solo se il
	sistema runtime di erlang non e' attivo, ma con i vari nodi già creati non ci dovrebbero
	essere problemi.

2)Sulla shell di ogni nodo eseguire 
	per ogni nodo creato al punto 1 -> net_kernel:connect_node(...nome nodo...). 
	cd("...percorso assoluto cartella dove c'e' il codice erlang....").

3)per ogni file .erl presente su quella cartella eseguire sulla shell di 1 nodo random
	c("NomeFile.erl").
