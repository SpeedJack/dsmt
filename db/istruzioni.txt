Andare nella cartella dove è installato glassfish
e usare il comando

$ ./asadmin

Dopodichè si apre la console di amministrazione di glassfish

asadmin>

Digitare il comando

asadmin> start-database

Adesso dopo aver avviato il db su IntellijIDEA potete aggiungere
una data source dal pannello database (solitamente a sinistra)
Scegliete Apache Derby

Nel pannello che vi si apre cambiate le impostazioni in
(sono link ma cliccandoci si apre un menu a tendina)

connection-type: default -> URL-only
driver: embedded -> remote

inserite nell'url:
jdbc:derby://localhost:1527/DSMT;create=true

click su OK e avete creato il db

dopo aver creato lo schema potete creare le tabelle:
tasto dx -> Run SQL Script...
cercate lo script nella cartella db (generate_schema.sql)
